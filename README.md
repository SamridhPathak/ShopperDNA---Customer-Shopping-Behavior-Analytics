# ShopperDNA - Customer Shopping Behavior Analytics

## 📌 Project Overview
**ShopperDNA** is a data analytics project designed to analyze customer purchasing patterns, subscription behavior, and revenue distribution across different categories and age groups.  

The project uses **PostgreSQL** as the database, **Python** for data analysis and database connectivity, and **Power BI** for interactive data visualization.

---

## 🎯 Objectives
- Analyze customer subscription trends  
- Understand revenue and sales distribution by category  
- Study customer behavior across different age groups  
- Build an interactive and insightful dashboard using Power BI  

---

## 🛠️ Tech Stack
- **Database:** PostgreSQL  
- **Programming Language:** Python  
- **Visualization Tool:** Power BI  
- **Libraries Used:**  
  - `psycopg2` / `sqlalchemy` – Database connection  
  - `pandas` – Data analysis and transformation  
  - `numpy` – Numerical operations  

---

## 🔄 Project Workflow
1. **Data Storage**
   - Customer data stored in PostgreSQL database.

2. **Database Connection**
   - Python is used to connect PostgreSQL with Power BI.
   - SQL queries extract required data.

3. **Data Analysis (Python)**
   - Data cleaning and preprocessing
   - Aggregations (average rating, revenue, sales, customer counts)

4. **Data Visualization (Power BI)**
   - Interactive dashboard created using processed data.

---

## 📈 Dashboard Features
- **KPIs**
  - Average Review Rating
  - Total Number of Customers
  - Average Purchase Amount

- **Visual Insights**
  - Subscription Status Distribution
  - Revenue by Category
  - Sales by Category
  - Revenue by Age Group
  - Sales by Age Group

- **Filters / Slicers**
  - Subscription Status
  - Gender
  - Product Category
  - Shipping Type
